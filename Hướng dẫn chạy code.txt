B1: Tải folder và mở project trong vscode

B2: Tạo môi trường : python -m venv .venv (Window)

B3: Activate môi trường .\.venv\Scripts\activate (Window)

B4: Tải thư viện:
+ pip install cvzone==1.5.6
+ pip install flask
+ pip install opencv-contrib-python==4.7.0.72
+ pip install opencv-python==4.7.0.72
+ pip install mediapipe
+ pip install openai

B5: Chọn interpretor bằng CTRL+P ghi ">Python: Select Interpretor" rồi chọn phương án có môi trường venv được recommended

B6: Cho phép môi trường :$env:FLASK_APP="main.py"

B7: Chạy chương trình flask --debug run hoặc chạy chương trình bằng nút run code của vscode 

B8: Copy link dẫn web lên google chrome

B9: Với mỗi trang web set tỉ lệ zoom của trang web phù hợp sao cho chữ không đè lên nhau và các chữ không bị che
